# XR

## What is it
**xr** is a command that will run any command you put in the config file.

## Configuring
Config file at $HOME/.config/xr/config.conf

**Config Format**
<command to be run>|<options to run that command>

**Example Config**
```text
vim ~/.vimrc|--cvim
```

## Usage
```bash
xr --cvim
```
